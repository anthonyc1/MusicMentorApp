Welcome to Music Mentor!

About

Music Mentor started as a project idea to create a program to help myself learn music theory. The idea grew from a python program to a Facebook Messenger Bot and now to an umbrella project consisting of an interactive messenger bot and this web application as you see here! This project seeks to help anybody, novice or veteran, to learn music theory. I hope this project will help you to either learn the basics or if not, be a useful online tool to help you figure out anything music theory related.

What you can do

So far, this web application supports:

1) scale identification

2) scale conversion
3) chord building
4) interval building

If you're not sure what any of that means, don't worry. Our application includes a quick and easy-to-understand tutorial to teach you the basics. Just click one of the buttons above to get started!
